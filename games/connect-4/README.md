# connect-4

Classic Connect Four. Drop discs into a 7x6 grid; first to line up four wins. Two modes: hot-seat vs another player, or single-player vs an AI opponent. Pixi renders the board and disc-drop animation; React renders all menus, HUD, and game-over overlay.

> Part of the [Games Collection](../../README.md) monorepo — built on shared [engine](../../packages/engine/README.md).

### Install dependencies

```bash
bun install
```

### Usage

```bash
moon run connect-4:dev        # http://localhost:3000
moon run connect-4:build      # production build
```
