# CLAUDE.md

Reference for developing games in this monorepo. Jump in when you need a specific topic.

- When adding a new topic, either extend an existing doc or add a new file and link it from this index.
- The root [`CLAUDE.md`](../CLAUDE.md) deliberately stays minimal and only points at `docs/` folder.
- You need to show short tokenomics table after each game creation / modification command (part - tokens). For reading part - decompose what you've read and how much tokens consumed each subsection.

| ------------------------------------- | ------------------------------------------------------------------- |
| [project.md](./docs/project.md)       | Monorepo layout, commands, conventions, build pipeline, deployment. |
| [game.md](./docs/game.md)             | Gameplay implementation — mental model, lifecycle, layout, events.  |
| [game-react.md](./docs/game-react.md) | React-UI game — events vs state, mount, theme, tweakpane reserve.   |
